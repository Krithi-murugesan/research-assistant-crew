# Research Assistant Crew (Production-Ready)

## Overview
A production-grade AI research pipeline built using CrewAI.  
The system dynamically researches any topic and generates both a detailed
markdown report and a concise executive summary.

## Architecture
- Modular agents
- Task factories
- Centralized configuration
- Structured logging
- CLI-driven execution

## How to Run
```bash
pip install -r requirements.txt
python main.py --topic "Retrieval-Augmented Generation"
